## Run it locally

Go into a root folder of your choice and run the following lines

`git clone https://github.com/DanceLife/One-MultiTenant.git` <br />
`cd One-MultiTenant` <br />
`npm install` <br />
`npm install firebase angularfire2 @types/firebase` <br />
`ng serve -o` <br />
